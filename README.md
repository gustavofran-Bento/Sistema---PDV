# 🧾 Sistema PDV (Ponto de Venda)

Sistema de ponto de venda desenvolvido com foco em simular operações reais de um ambiente comercial, permitindo o registro de vendas, controle de produtos e interação com banco de dados.

## 🚀 Tecnologias utilizadas
- HTML
- CSS
- JavaScript
- Node.js
- Banco de Dados

## 💡 Funcionalidades
- Registro de vendas em tempo real
- Controle de produtos
- Histórico de vendas
- Interface de caixa (PDV)
- Integração com banco de dados

## 🎯 Objetivo do projeto
Este projeto foi desenvolvido com o objetivo de aplicar na prática conceitos de desenvolvimento web, integração entre frontend e backend, e manipulação de dados, simulando um sistema utilizado em comércios reais.

## 📚 Aprendizados
- Estruturação de projetos fullstack
- Comunicação entre frontend e backend
- Manipulação de banco de dados
- Organização de código e boas práticas

🚧 Em desenvolvimento — melhorias e novas funcionalidades sendo implementadas.
